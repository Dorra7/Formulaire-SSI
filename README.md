# Rendu 1 - Sécurité des Systèmes d'Informations (SSI)

## Description

Ce projet est une interface de connexion simple réalisée dans le cadre du module SSI. Il permet de simuler une authentification et une gestion dynamique de champs d'identifiants.

## Fonctionnalités

- **Validation** : Vérifie les identifiants (Test : `admin` / `1234`).
- **Reset** : Réinitialise tous les champs du formulaire.
- **Ajout Compte** : Ajoute dynamiquement un nouveau champ de saisie d'identifiant.

## Réalisation Technique

- **Frontend** : HTML5 / CSS3 (Vanilla)
- **Logique** : JavaScript (Vanilla - sans framework)
- **Architecture** : Fichiers séparés pour une meilleure maintenance.
